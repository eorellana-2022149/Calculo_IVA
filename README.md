# Calculo_IVA
Actividad: Cálculo de IVA, Subtotal y Total con TypeScript 
 
El usuario puede ver un catálogo de productos, agregar productos al carrito, consultar un resumen de la compra y confirmar la compra. El sistema calcula automáticamente el subtotal, el IVA del 12% y el total a pagar.

El proyecto está organizado en varias carpetas para mantener el código ordenado:

Models: contiene las interfaces utilizadas para representar los productos y los elementos de una compra.
Data: almacena el catálogo de productos disponibles.
Services: contiene las funciones que realizan los cálculos y administran las compras.
Index: es el archivo principal que interactúa con el usuario mediante un menú en consola.

Funcionalidades:
Ver el catálogo de productos.
Agregar productos al carrito.
Calcular automáticamente el subtotal.
Calcular el IVA del 12%.
Calcular el total de la compra.
Mostrar un resumen de la compra actual.
Confirmar la compra.
Vaciar el carrito cuando sea necesario.
Dependencias utilizadas

Este proyecto utiliza:
TypeScript
Node.js
PNPM
Prompt-Sync
ts-node
ts-node-dev

Ejemplo de funcionamiento:

El usuario puede seleccionar una opción del menú, agregar productos al carrito y consultar un resumen de la compra. El sistema muestra el subtotal, el IVA y el total a pagar.
